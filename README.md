# Fsociety Tool
Fsociety Tool is a mini tool with DDoS,ip locate and VPN test function.

# YOU WILL NEED
python with pip (python must be 3.x.x)

git

psutil

# INSTALLATION FOR WINDOWS
git clone https://github.com/Mixyy2000/Fsociety-Tool.git

cd FsocietyTool

pip install -r requirements.txt

python FsocietyTool.py

# INSTALLATION FOR LINUX

git clone https://github.com/Mixyy2000/Fsociety-Tool.git

cd FsocietyTool

pip install -r requirements.txt

ls

python3 FsocietyTool.py
